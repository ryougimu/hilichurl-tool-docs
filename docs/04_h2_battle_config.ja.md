# 4_h2 戦闘設定

![戦闘設定](images/04/hero/2.png)

戦闘設定タブでは、キャラクターの攻撃方式と装備を選択します。

## 攻撃モード（`editor.section_attack_mode`）

| フィールド | 説明 |
|-----------|------|
| 基本攻撃（`editor.basic`） | キャラクターの基本攻撃方式 |
| チャージ攻撃（`editor.charge`） | キャラクターのチャージ攻撃方式 |
| 元素スキル（`editor.skill`） | キャラクターの元素スキル |
| 元素爆発（`editor.burst`） | キャラクターの元素爆発 |

各項目は**テンプレートを選択**（`editor.select_template`）ボタンでダイアログから選びます。

## その他の設定（`editor.section_other_config`）

| フィールド | 説明 |
|-----------|------|
| 装備（`editor.item`） | キャラクターが携帯する装備 |

## カスタムインポート

元素スキル、元素爆発、装備の3項目はカスタムインポートに対応しています。既存テンプレートで満たせない場合、自分で書いたスクリプトをインポートできます。**カスタムをインポート**（`editor.import_custom`）ボタンでファイルを選択します。隣の **?** ボタンでカスタム作成ガイドを参照できます。
