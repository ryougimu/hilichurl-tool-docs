# 第四章：Hero 编辑

选中一个 Hero 角色后，编辑区会显示以下标签页：

| 标签页 | 用途 | 详细说明 |
|--------|------|----------|
| 基本信息 | 名称、属性、战斗图标 | [04_h1](04_h1_basic_info.md) |
| 战斗配置 | 攻击方式、技能、携带物品 | [04_h2](04_h2_battle_config.md) |
| 资源 | 图片、音频资源管理 | [04_h3](04_h3_resource.md) |
| 登场方式 | 出场条件、登场位置、随从 | [04_h4](04_h4_appearance.md) |
| 事件 | 剧情事件编排 | [04_h5](04_h5_event_editor.md) |
| 其他 | 物品掉落概率 | [04_h6](04_h6_other.md) |

所有修改会自动保存，无需手动操作。
