# 第六章：国际化

编辑器支持三种界面语言：

| 语言 | 代码 |
|------|------|
| 中文 | chs |
| English | en |
| 日本語 | jp |

## 切换语言

语言切换入口在欢迎页右上角的下拉菜单中。切换后所有界面文字会立即更新。

语言设置会自动保存，下次启动编辑器时沿用上次的选择。

## 翻译覆盖范围

编辑器的国际化分为两部分：

**界面翻译** —— 编辑器自身的菜单、按钮、标签等文字，切换语言后立即生效。

**数据翻译** —— 下拉选项中显示的游戏数据名称（英雄名、物品名、技能名等），随语言切换自动更新。部分数据翻译可能不完整，未翻译的项会显示原始英文 ID。

## 使用说明文档

编辑器内置的操作指南（通过帮助按钮查看）也支持多语言，位于 `options/md/` 目录。
