# 4_h6 その他

![その他](images/04/hero/7.png)

その他タブは現在**売春宿アイテム**（`other.section_brothel`）のドロップ確率を管理するために使用されます。

複数のドロップアイテムを追加できます。各アイテムには次が含まれます。

- **アイテム**（`other.select_item`）— アイテムリストから選択（アイコンと名前を表示）  
- **確率** — ドロップ確率値を設定  

下部のセレクターでアイテムを選択し確率を設定した後、**アイテム追加**（`other.add_item`）をクリックして追加します。追加されたアイテムは個別に削除できます。
