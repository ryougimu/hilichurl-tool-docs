## 既知の制限

現在のバージョンには次の制限があります。今後のバージョンで改善予定です。

- イベントカテゴリーとサブイベントはプリセット、カスタム作成不可  
- CG と背景ブロックはビジュアルインターフェースで編集不可（回避策：[ソース編集](04_h5_event_editor.md#source-editing)）  
- 一部のデータ翻訳が不完全、未翻訳項目は元の英語 ID を表示  
- 登場条件データはゲームから自動抽出されており、完全には正確でない可能性あり  
- サブヒーロー機能  
- カスタムオプション  
- 新世界対応（少なくとも編集可能）

## よくある質問

### 起動時に設定ダイアログが表示される？

初回起動または設定ファイル欠損時に自動表示されます。Options ディレクトリと Godot 実行ファイルパスの設定が必要です。[クイックスタート](02_quick_start.md)を参照してください。

### エクスポート時に Godot パス未設定と表示される？

**設定**で Godot 実行ファイルパスを指定してください。まだ PCK パックが不要なら、「PCKとしてエクスポート」をオフにして散在ファイルを直接エクスポートできます。

### キャラクター ID を間違えた？

キャラクター ID は作成後変更できません。変更するには、新しいキャラクターを作成してデータを再設定する必要があります。

### イベントの CG/背景を編集できない？

現在のバージョンでは、CG と背景は基礎ブロックとしてマークされており、ビジュアルインターフェースでは編集できません。[ソース編集](04_h5_event_editor.md#source-editing)で変更してください。

### エクスポートディレクトリのファイルが消えた？

各エクスポート前に `export/` ディレクトリは自動クリアされます。手動変更したファイルを保持する必要がある場合は、エクスポート前に別の場所にコピーしてください。

### 言語切り替え後も一部の選択肢が英語のまま？

データ翻訳が不完全な可能性があり、未翻訳項目は元の英語 ID が表示されます。

### 登場条件がゲーム内で機能しない？

条件データはゲームから自動抽出されており、完全には正確でない可能性があります。問題があれば報告してください。

## 用語集

| 用語 | 説明 |
|------|------|
| Options | エディタの基礎データパッケージ、ヒーローテンプレート・アイテム・効果音などのゲームデータを含む |
| PCK | Godot リソースパック形式、複数ファイルを1つにパック |
| Install スクリプト | エクスポート時に生成される GDScript ファイル、ゲーム読み込み時に実行されキャラクターデータを登録 |
| ブロック（Block） | イベント内の最小命令単位、1つのセリフや1つの背景切り替えなど |
| ページ区切り | イベント内の「クリック待ち」ブロック、イベントを複数ページに分割、ゲーム内でプレイヤークリックで続行 |
| 基礎ブロック | イベントテンプレートがプリセットしたブロック、追加/編集/削除不可、移動のみ可 |
| バリアント | 同じ画像の異なる変種（異なる表情状態など）、エディタはサムネイルを自動生成可 |
| event_assets | プロジェクトレベル共有リソースディレクトリ、全キャラクターが背景・立ち絵・ボイスを共有可 |
| ソース編集 | テキスト形式でイベントブロックコマンドを直接編集、ビジュアルインターフェースの制限を回避 |
