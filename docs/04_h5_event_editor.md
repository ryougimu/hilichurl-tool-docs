# 4_h5 事件编辑器

![事件编辑器](images/04/hero/5.png)

事件编辑器用于编排角色的剧情事件。每个事件由一系列"块"组成，块按顺序排列，游戏中会依次执行。

## 界面布局

事件编辑器分为左右两个区域：

- **左侧：事件导航树** —— 按类别列出所有可编辑的事件
- **右侧：编辑与预览区** —— 左右并排显示预览面板和块编辑面板

预览面板和块编辑面板之间可通过拖拽调整宽度比例。

## 事件导航

左侧树形菜单按类别组织事件：

部分事件按角色状态区分多个子事件，状态编号含义如下：

| 编号 | 状态 |
|------|------|
| a | 反抗（Disobedient） |
| b | 顺从（Submissive） |
| c | 绝望（Helpless） |
| d | 淫乱（Corrupted） |

| 类别 | 包含的事件 |
|------|-----------|
| 登场（Appeared） | 出现事件 |
| 俘获（Captured） | 捕获事件 |
| 出产（Give Birth） | a / b / c / d 四种状态 |
| 祭坛恶堕（Altar） | 1=开始 / 2=结束 |
| 调教（Torture） | 普通 / 淫堕 / 碎眼 |
| 妓院（Brothel） | a / b / c / d 四种状态（见下方特殊说明） |
| 新娘（Wedding） | a / b / c / d 四种状态 |
| 新娘SP（Wedding2） | a / b / c / d 四种状态 |

点击事件名称即可打开编辑。

### 事件特殊说明

**基础块（base）**

大部分事件包含预设的基础块（如 CG、Background），这些块由事件模板自动生成，不可删除但可以移动位置。

**前置资源（frontload）**

除 Appeared 外，其他事件都需要对应的 CG 图片资源。每个事件要求的 CG 图片编号不同（如捕获事件需要 01 系列、出产事件需要 03 系列等）。如果缺少必需的 CG 图片，工具栏会显示 ⚠ 警告，将鼠标悬停在警告上可以查看缺失的具体文件名。

**附加项（extra）**

部分事件类别（如 Wedding2/新娘SP）需要设置额外信息。在事件导航树中，这类类别名称旁会显示 ✎ 编辑按钮。

目前支持的附加项为**头衔设置**——击败该角色后玩家获得的头衔名称，需填写三语言翻译（中文/英文/日文）。未设置时类别名后会显示 ⚠ 提示。启用该类别下的事件后，头衔翻译会在导出时写入游戏数据。

**妓院事件特殊说明**

妓院中仅允许放入绝望(c)、淫乱(d)、面具状态的角色。但面具状态较为特殊——游戏运行时会走当前实际状态的逻辑（反抗/顺从/绝望/淫乱都有可能），因此保留了编辑权限，最少可以仅编辑淫乱(d)。

### 启用/禁用事件

每个事件右上方有 **Enable** 开关。只有启用的事件才会在导出时包含。

### 资源警告

部分事件需要对应的 CG 图片资源。如果缺少必需的图片，事件名称旁会显示 **⚠** 警告。你需要在资源标签页中导入对应的 CG 图片后警告才会消失。

## 块列表

右侧面板显示当前事件的所有块，每个块以彩色卡片形式展示。

### 查看模式

块列表有两种查看模式，通过顶部按钮切换：

- **全部模式** —— 显示事件中的所有块
- **分页模式** —— 以"等待点击"块为分页边界，每页显示一段内容。预览面板和块编辑面板各有独立的翻页导航，可分别翻页

### 块操作

- **添加块** —— 点击「Add Block」按钮，从菜单中选择块类型
- **编辑块** —— 点击块卡片，弹出编辑窗口填写内容
- **删除块** —— 点击块卡片右上角的 ✕ 按钮
- **拖拽排序** —— 按住块卡片左侧的 ≡ 手柄上下拖动
- **源码编辑** —— 详见下方[源码编辑](#source-editing)

部分块由事件模板预设，属于基础块，不可删除但可以移动位置。

## 块类型

共 18 种块类型，分为以下几类：

### 对话

| 块类型 | 游戏指令 | 作用 |
|--------|---------|------|
| 说话人 | `name:` | 设置当前说话人名字，留空可清除 |
| 对话 | `dialog:` | 显示对话框文本 |

### 画面

| 块类型 | 游戏指令 | 作用 |
|--------|---------|------|
| 背景 | `background:` | 切换背景图 |
| CG | `CG:` | 显示/清除 CG 图片（叠加在背景上方） |
| 设置立绘 | `figure1:` / `figure2:` | 在右侧(1)或左侧(2)放置立绘，留空则显示当前角色立绘 |
| 隐藏立绘 | `hide_figure1:` / `hide_figure2:` | 移除指定位置的立绘 |
| 画面变黑 | `turn_black:` | 黑屏遮罩，直到下一个背景块恢复 |
| 震动 | `shake:` / `shake_x:` | 画面震动，可选纵向或横向，可调强度 |

### 音频

| 块类型 | 游戏指令 | 作用 |
|--------|---------|------|
| BGM | `bgm:` | 切换背景音乐，留空停止播放 |
| 音效 | `sound:` | 播放一次性音效，可调音量 |
| 语音 | `voice:` | 播放自定义语音文件（Voice/ 目录），可调音量 |
| 预设语音 | `preset_voice:` | 播放预设语音关键字，可调音量 |
| 循环语音 | `continue_voice:` | 开始循环播放语音，可调音量和速度 |
| 停止循环语音 | `continue_voice_end:` | 停止循环语音 |
| 循环音效 | `continue_sound:` | 开始循环播放音效，可调音量和速度 |
| 停止循环音效 | `continue_sound_end:` | 停止循环音效 |

voice 主要是角色语音，需要事先导入。在本工具中当前实现方式本质，voice和preset_voice无本质区别。

### 控制

| 块类型 | 游戏指令 | 作用 |
|--------|---------|------|
| 延迟 | `auto:` | 等待指定秒数后自动推进，玩家点击可跳过 |
| 等待点击 | （空行） | 暂停等待玩家点击，同时作为分页边界 |

> **注意**：游戏引擎还支持 `hide_background`、`signal`、`icon`/`end_icon` 等指令，编辑器暂未提供对应的块类型。如需使用这些指令，可通过源码编辑模式手动输入。

## 编辑块内容

点击块卡片后弹出编辑窗口，根据块类型显示不同的编辑项：

- **文本类**（说话人/对话）—— 输入框或多行文本框
- **图片类**（背景/CG/立绘）—— 图片选择器，左侧列表右侧预览，支持导入
- **音频类**（BGM/音效/语音）—— 音频选择器，支持搜索、试听和导入
- **数值类**（音量/速度/震动强度）—— 数字滑块
- **选项类**（立绘位置/震动方向）—— 下拉菜单

## 预览面板

中央预览面板实时渲染事件效果，包括：

- 背景图和 CG 的叠加显示
- 左右两侧的立绘
- 对话框（说话人名字 + 对话文本）
- 当前 BGM 名称
- 黑屏遮罩效果

在分页模式下，预览会根据当前页码累积状态——前页设置的背景、立绘等会保留到后续页面。

## 资源来源

事件中使用的图片和音频资源按以下优先级查找：

1. **角色目录** —— 当前角色的 Figure/CG 等文件夹
2. **项目共享** —— 项目的 event_assets 目录（所有角色共用）
3. **全局预设** —— options 配置包中的预设资源

通过编辑窗口中的导入按钮导入的资源会自动存放到项目共享目录中。

## 源码编辑 {#source-editing}

点击工具栏的「Source Edit」按钮，可打开文本编辑窗口直接编辑事件的原始指令格式。

![源码编辑](images/04/hero/6.png)

### 为什么需要源码编辑？

当前版本中，CG 和背景等块被标记为基础块，在可视化界面中不可编辑或删除（这是设计限制，后续版本会优化）。如果你需要自定义这些内容，源码编辑是目前的唯一方式。

此外，游戏引擎支持一些编辑器尚未提供可视化支持的指令（如 `hide_background`、`signal`、`icon`/`end_icon`），也可以通过源码编辑手动输入。

### 格式说明

每行一条指令，格式为 `指令:参数`。空行表示等待点击（分页）。

```
background:bg_forest.webp
name:Alice
dialog:这里是哪里...

CG:anna_01_1.webp
shake:5
name:Alice
dialog:发生了什么?!

```

- `name:` 设置说话人，`dialog:` 显示对话文本，两者是独立指令
- 带多个参数的指令使用空格分隔的键值对：`sound:sound=slash_1 vol=0`
- 支持模板变量：`{{heroine}}` 会在游戏中替换为角色类型名

### 注意事项

- 源码编辑直接操作原始数据，格式错误可能导致事件无法正常运行
- 源码中引用的资源文件需要自行确保已存在，编辑器不会自动导入。各类资源对应的目录：
  - 背景图：`{项目}/event_assets/Background/` 或 options 预设
  - 立绘：`characters/{角色ID}/Figure/` 或 `{项目}/event_assets/Figure/`
  - CG：`characters/{角色ID}/CG/`
  - 语音：`characters/{角色ID}/Voice/` 或 `{项目}/event_assets/Voice/`
  - BGM/音效：来自 options 配置包（`audio/bgm/`、`audio/sound/`）
- 保存后可视化界面会同步更新，可以通过预览面板验证效果
