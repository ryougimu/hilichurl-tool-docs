# 第6章：国際化

エディタは3つの UI 言語に対応しています。

| 言語 | コード |
|------|--------|
| 中文 | chs |
| English | en |
| 日本語 | jp |

## 言語の切り替え

言語切り替えはウェルカム画面右上のドロップダウンにあります。切り替え後、全 UI テキストがすぐに更新されます。

言語設定は自動保存され、次回起動時に前回の選択が使用されます。

## 翻訳カバー範囲

エディタの国際化は2つの部分があります。

**UI 翻訳** — エディタ自体のメニュー、ボタン、ラベルなどのテキスト。言語切り替え後すぐに有効になります。

**データ翻訳** — ドロップダウンに表示されるゲームデータ名（ヒーロー名、アイテム名、スキル名など）。言語切り替えで自動更新されます。一部のデータ翻訳は不完全な場合があり、未翻訳の項目は元の英語 ID が表示されます。

## 利用ガイド

内蔵の操作ガイド（ヘルプボタンからアクセス）も多言語対応しています。`options/md/` ディレクトリにあります。
