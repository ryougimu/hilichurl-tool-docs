# 第1章：エディタについて

## これは何か

シンプルなゲーム用 Mod エディタです。現時点では新規ヒーローの作成と、一部のストーリーイベントの編集に対応しています。

開発は継続中で、今後さらに機能が増えていく予定です。

## できること

- **ヒーロー編集** — キャラクターの基本情報・戦闘ステータス・リソース・イベントなどを設定  
- **プロジェクト管理** — 複数の独立したプロジェクトを作成・管理し、各プロジェクトに複数キャラクターを持てる  
- **エクスポート／パック** — キャラデータとリソースをゲームが読み込める形にまとめる  

詳しくは[クイックスタート](02_quick_start.md)以降の章を参照してください。

## 全体の流れ

典型的な作業の流れは次のとおりです。

```
準備                    制作                      公開
───────                ───────                   ───────
options を入手    →    プロジェクト作成     →    エクスポート／パック
Mod 用素材準備         キャラ追加                パック方式の選択
（立ち絵／CG／音声）   パラメータ編集
                       イベント構成
                       素材インポート
```

- **準備**：options パッケージを入手し、Mod 用の素材（立ち絵、CG、ボイスなど）を用意します。options にはヒーローテンプレート、アイテム一覧、効果音ライブラリなど、エディタが参照する基礎データが含まれます。詳しくは[基礎データ（Options）](03_options_source.md)。  
- **制作**：日常的な作業です。プロジェクト作成、キャラ追加、各種データの編集、素材のインポートを行います。  
- **公開**：完成したキャラデータをゲームが解釈できる形で出力します。[エクスポートとパック](05_export.md)を参照。

## 画面の概要

主に次の2画面があります。

### ウェルカム画面

![ウェルカム画面](images/01/1.png)

起動直後に表示されます。最近開いたプロジェクトの一覧があります。

- プロジェクト名をクリックして開く  
- **新規**でプロジェクト作成  
- **開く**で既存フォルダを指定  
- 表示言語の切り替え（中文 / English / 日本語）

初回起動時は、**Options ディレクトリ**と **Godot 実行ファイル**の2つを設定する必要があります。あとから**設定**で変更できます。

### メインウィンドウ

![メインウィンドウ](images/01/2.png)

プロジェクトを開いたあとの画面です。大きく3領域に分かれます。

- **左：キャラクター一覧** — プロジェクト内の全キャラを表示。検索・折りたたみに対応  
- **中央：編集エリア** — タブで各機能を切り替え  
- **下部：ログ** — 保存やインポートなどの結果を確認

上部には**プロジェクト**（新規／開く／閉じる）、**ツール**、**パック**、**設定**があります。

## 設計上のメモ

**自動保存** — 変更は自動で保存されます。手動の保存ボタンはありません。

**オプション駆動** — イベント種別や選択肢リストなどは options パッケージ由来です。パッケージを更新するとエディタ側の選択肢も追従します。
