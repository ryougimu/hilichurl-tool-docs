# 4_h6 Other

![Other](images/04/hero/7.png)

The Other tab is currently used to manage **Brothel item drop rates** (`other.section_brothel`).

You can add multiple drop items. Each item includes:

- **Item** (`other.select_item`) — Choose from the item list (shows icon and name)  
- **Rate** — Set the drop rate value  

After selecting an item and setting the rate in the bottom selector, click **Add Item** (`other.add_item`) to add it. Added items can be deleted individually.
