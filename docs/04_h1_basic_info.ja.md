# 4_h1 基本情報

![基本情報](images/04/hero/1.png)

基本情報タブは左右2つのエリアに分かれています。

## 左：属性フォーム

フォームは3つのグループに分かれています。

**名前**

- 中国語名（`editor.name_chs`） / 英語名（`editor.name_en`） / 日本語名（`editor.name_jp`）

**キャラクター属性**（`editor.section_character`）

| フィールド | 説明 |
|-----------|------|
| 星ランク（`editor.star`） | キャラクターの星ランク、各種属性に影響 |
| リーダーシップ（`editor.leader_value`） | 味方ユニット時のリーダーシップ |
| 子宮容量（`editor.womb`） | 一度に産出されるユニット数に影響 |
| 基礎攻撃（`editor.attack_base`） | 味方ユニット時の攻撃力ボーナス |
| 基礎防御（`editor.defend_base`） | 味方ユニット時の防御力ボーナス |
| 基礎HP（`editor.hp_base`） | 味方ユニット時のHPボーナス |
| 元素（`editor.element`） | キャラクターの元素属性 |
| コアタイプ（`editor.core`） | キャラクターのコアタイプ |
| 神の眼タイプ（`editor.vision`） | キャラクターの神の眼タイプ |

**戦闘属性（敵ユニット）**（`editor.section_combat`）

| フィールド | 説明 |
|-----------|------|
| 攻撃力（`editor.attack`） | 敵ユニットの攻撃力 |
| 防御力（`editor.defend`） | 敵ユニットの防御力 |
| HP（`editor.hp`） | 敵ユニットのHP |
| 速度（`editor.speed`） | 戦闘中の移動範囲に影響 |
| 先制値（`editor.initiative`） | 戦闘中の行動順序に影響 |
| 移動方式（`editor.move`） | 戦場でのユニットの移動方式 |
| 捕獲率（`editor.catch_rate`） | 基礎捕獲率 |
| 戦力（`editor.value`） | 自動戦闘評価用の基礎戦闘力 |
| Boss行動（`editor.boss_behavior`） | 敵として使用時のAI行動パターン |
| 元素耐性（`editor.res`） | 自身の所属元素に対する耐性 |

## 右：戦闘アイコンプレビュー

現在のキャラクターの戦闘アイコンを表示します。複数のタブで異なる状態のアイコン（デフォルト、疲労、特殊状態など）を切り替えられます。

画像のインポート方法：

- プレビューエリアをクリックしてファイルを選択  
- 画像ファイルをプレビューエリアに直接ドラッグ&ドロップ

インポート後、エディタは自動的に異なるサイズのバリアントを生成します。
