# Chapter 6: Internationalization

The editor supports three UI languages:

| Language | Code |
|----------|------|
| 中文 | chs |
| English | en |
| 日本語 | jp |

## Switching language

The language switcher is in the welcome screen's top-right drop-down. After switching, all UI text updates immediately.

Language settings auto-save; the editor uses your last choice on next launch.

## Translation coverage

The editor's internationalization has two parts:

**UI translation** — The editor's own menus, buttons, labels, etc. Takes effect immediately after switching language.

**Data translation** — Game data names shown in drop-downs (hero names, item names, skill names, etc.) auto-update when you switch language. Some data translations may be incomplete; untranslated items show their original English IDs.

## User guide

The built-in operation guide (accessed via help button) also supports multiple languages. Located in the `options/md/` directory.
