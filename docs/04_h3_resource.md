# 4_h3 资源

![资源](images/04/hero/3.png)

资源标签页用于管理角色的图片和音频资源，采用三栏布局：

- **左栏：分类树** —— 按类别展示资源分组
- **中栏：资源列表** —— 显示该分类下的所有资源项及其完成状态
- **右栏：预览** —— 预览选中的资源，支持差分查看

资源可以通过拖拽文件到预览区域来导入。

**你只需配置需要的资源，不必全部填满。**

## 资源分类

| 分类 | 说明 | 格式 |
|------|------|------|
| 立绘（Portrait） | 角色立绘，包含多种表情状态 | PNG |
| 交配动画（Mating） | 交配场景图，包含拖拽和各怪物类型 | PNG |
| 事件 CG（CG） | 剧情事件 CG 图片 | PNG |
| 刑具（Torture） | 刑具相关图片 | PNG |
| 语音（Voice） | 角色战斗语音 | OGG |

## 自动生成

立绘和交配动画类别在导入一张大图后，会自动生成一张对应的缩略图（存放在 Animated 文件夹）。你只需导入原始大图，缩略图无需手动制作。
