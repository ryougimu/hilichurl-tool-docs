# 第4章：ヒーロー編集

Hero キャラクターを選択すると、編集エリアに次のタブが表示されます。

| タブ | 用途 | 詳細 |
|------|------|------|
| 基本情報 | 名前、属性、戦闘アイコン | [04_h1](04_h1_basic_info.md) |
| 戦闘設定 | 攻撃方式、スキル、装備 | [04_h2](04_h2_battle_config.md) |
| リソース管理 | 画像・音声リソースの管理 | [04_h3](04_h3_resource.md) |
| チーム設定 | 登場条件、位置、ミニオン | [04_h4](04_h4_appearance.md) |
| イベント | ストーリーイベントの構成 | [04_h5](04_h5_event_editor.md) |
| その他 | アイテムドロップ確率 | [04_h6](04_h6_other.md) |

すべての変更は自動保存され、手動操作は不要です。
