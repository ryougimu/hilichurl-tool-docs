# Chapter 4: Hero editing

When you select a Hero character, the editor area shows these tabs:

| Tab | Purpose | Details |
|-----|---------|---------|
| Basic Info | Name, attributes, combat icon | [04_h1](04_h1_basic_info.md) |
| Battle Config | Attack modes, skills, equipment | [04_h2](04_h2_battle_config.md) |
| Resources | Image and audio resource management | [04_h3](04_h3_resource.md) |
| Team | Appear conditions, location, minions | [04_h4](04_h4_appearance.md) |
| Events | Story event arrangement | [04_h5](04_h5_event_editor.md) |
| Other | Item drop rates | [04_h6](04_h6_other.md) |

All changes auto-save; no manual action needed.
