# 4_h2 战斗配置

![战斗配置](images/04/hero/2.png)

战斗配置标签页用于选择角色的攻击方式和携带装备。

## 攻击模式

| 字段 | 说明 |
|------|------|
| 基础攻击（Basic Attack） | 角色的基础攻击方式 |
| 蓄力攻击（Charge Attack） | 角色的蓄力攻击方式 |
| 元素战技（Elemental Skill） | 角色的元素战技 |
| 元素爆发（Elemental Burst） | 角色的元素爆发 |

每项通过「Select Template」弹窗从模板中选择。

## 其他配置

| 字段 | 说明 |
|------|------|
| 装备（Equipment） | 角色携带的装备 |

## 自定义导入

元素战技、元素爆发、装备三项支持自定义导入——如果现有模板不能满足需求，你可以导入自己编写的脚本。点击「Import Custom」按钮选择文件导入，旁边的「?」按钮可查看自定义编写指南。
