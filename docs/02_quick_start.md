# 第二章：快速开始

## 首次启动

启动编辑器后，首先看到欢迎页。首次使用时，编辑器会自动弹出设置对话框，要求配置两个路径：

![设置对话框](images/02/1.png)

- **Options 目录** —— 编辑器的基础数据配置包所在目录。你需要先下载 options 配置包，解压后在此指定。详见[基础数据来源](03_options_source.md)。
- **Godot 可执行文件路径** —— 导出 PCK 打包时需要使用的 Godot 引擎路径。如果暂时不需要导出功能，可以稍后再配置。

这些设置可以随时通过主界面顶部的设置按钮修改。

## 创建项目

点击欢迎页右上角的「新建」按钮，在弹出的文件夹选择器中选择一个空文件夹作为项目目录。编辑器会在该目录下生成项目结构文件，并自动进入主界面。

项目目录名称即为项目名称。

## 添加角色

进入主界面后，如果项目中还没有角色，编辑区会显示空白提示。

点击左侧角色侧边栏标题栏右侧的「+」按钮：

![添加角色对话框](images/02/2.png)

- **角色 ID** —— 角色的唯一标识符，仅支持英文字母和数字。这个 ID 将用于文件夹命名和数据导出，创建后无法更改。
- **英雄模板**（可选） —— 从已有的游戏英雄中选择一个作为模板，编辑器会自动填充该英雄的基础属性和战斗配置。你也可以不选择模板，从零开始配置。

确认后，角色出现在左侧列表中，编辑区自动加载该角色的编辑界面。

## 编辑角色

编辑区包含多个标签页，每个标签页负责角色的一个方面：

| 标签页 | 用途 |
|--------|------|
| 基本信息 | 角色名称、属性、稀有度等基础数据，以及战斗图标、头像预览 |
| 战斗配置 | 选择攻击方式、技能和携带物品 |
| 资源 | 浏览和导入角色的图片、音频资源 |
| 登场方式 | 出场条件、登场位置、随从配置 |
| 事件 | 编排角色的剧情事件 |
| 其他 | 物品掉落概率等额外设置 |

所有修改会自动保存，无需手动操作。

各标签页的详细说明见[Hero 编辑](04_hero_editing.md)。

## 切换和管理角色

- **切换角色** —— 点击左侧列表中的角色名称即可切换
- **搜索角色** —— 在侧边栏顶部的搜索框输入关键字过滤
- **删除角色** —— 点击角色名称右侧的删除按钮（会弹出确认对话框）
- **折叠侧边栏** —— 点击侧边栏的折叠按钮可以隐藏角色列表，腾出更多编辑空间

## 切换语言

编辑器支持三种界面语言：中文、English、日本語。

语言切换入口在欢迎页右上角的下拉菜单中。切换后，所有界面文字会立即更新。

## 项目管理

通过主界面顶部的「项目」菜单，你可以：

- **新建项目** —— 创建一个全新的项目
- **打开项目** —— 打开一个已有的项目文件夹
- **关闭项目** —— 关闭当前项目，返回欢迎页

欢迎页会记住你最近打开过的项目，方便快速访问。如果项目文件夹被移动或删除，列表中该项会显示为失效状态。
